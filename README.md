# recaptcha-gen
reCaptcha梗图生成器

## 使用
[在线体验](https://latedream.cn/recaptcha-gen/)或者
```bash
git clone https://github.com/LateDreamXD/recaptcha-gen.git
cd recaptcha-gen
pnpm i && pnpm build && pnpm preview
```

## 开发
```bash
git clone https://github.com/LateDreamXD/recaptcha-gen.git
cd recaptcha-gen
pnpm i && pnpm dev
```

## 技术栈
- 主要开发语言: Vue3, TypeScript, SCSS
- 框架: Vite
- 图片生成: dom-to-image
- 使用的样式表: Pico CSS, Google reCaptcha 样式表
